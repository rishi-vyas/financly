# Financly – AI-powered Personal Finance Platform

## Problem
Managing personal finances is complex and lacks real-time actionable insights.

## Solution
Built a real-time financial analytics platform that tracks income, expenses, and trends.

## Product Thinking
Helps users make smarter financial decisions through data-driven insights.

## Data & Insights
•⁠  ⁠Real-time transaction tracking
•⁠  ⁠Aggregated financial analytics
•⁠  ⁠Trend visualization

## AI Integration
Used LLMs to generate personalized financial insights based on user spending behavior.

## Tech Stack
ReactJS, Firebase, JavaScript, APIs

## Impact
Improved visibility into spending habits and enabled better financial planning.
